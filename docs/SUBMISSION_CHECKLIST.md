# Submission readiness checklist

Use this before journal submission.

## Data and variables

- [ ] Firm-year panel has valid firm identifier and year.
- [ ] Corporate diversification is coded so higher values mean more diversification.
- [ ] Amihud illiquidity is calculated from daily trading data.
- [ ] Earnings quality is coded so higher values mean better earnings quality.
- [ ] Governance index is constructed and CEO duality is reversed.
- [ ] All continuous variables are winsorized or outliers are addressed.
- [ ] Industry and year definitions are documented.

## Empirical results

- [ ] H1 direct effect table completed.
- [ ] H2 first-stage information asymmetry table completed.
- [ ] H3 information asymmetry to earnings quality table completed.
- [ ] H4 mediation effect tested.
- [ ] H5 moderation interaction tested.
- [ ] H6 conditional indirect effects table completed.
- [ ] Robustness checks completed.
- [ ] Machine-learning robustness included only as supplementary/predictive evidence.

## Manuscript

- [ ] Replace all placeholder text.
- [ ] Insert final regression tables.
- [ ] Cite all tables and figures in the text.
- [ ] Verify every reference exists and is accurate.
- [ ] Check journal formatting requirements.
- [ ] Confirm data availability statement.
- [ ] Confirm author names and affiliations.
- [ ] Get advisor/coauthor approval before submission.
